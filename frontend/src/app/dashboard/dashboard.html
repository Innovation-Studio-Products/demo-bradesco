Arquitetura da POC

A POC foi estruturada usando tecnologias modernas de ponta a ponta:

  • Frontend: Angular v20+ com Standalone Components, Signals para estado e Router Guards funcionais.
  • Backend: Python com FastAPI, tipagem estática do Pydantic e servidor Uvicorn.
  • Banco de Dados: SQLite embarcado (arquivo local `users.db`), integrado via biblioteca interna `sqlite3` sem dependências pesadas.
  • Comunicação: REST API JSON e proteção via token de autorização.
